Installation über npm:
npm install handlebars -g

Nach Änderungen in den HTML-Dateien auf Kommandozeile ausführen: 
handlebars ./de -a -b -e html -f ../www/app/handlebars/de.js
handlebars ./de-de -b -a -e html -f ../www/app/handlebars/de-de.js
handlebars ./en -a -b -e html -f ../www/app/handlebars/en.js
handlebars ./all -a -b -e html -f ../www/app/handlebars/all.js
